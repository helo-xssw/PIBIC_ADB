## Coleta de Dados de Redes Móveis com ADB

#### Descrição
Este repositório apresenta o desenvolvimento de um projeto PIBIC voltado ao uso do Android Debug Bridge (ADB) para coleta de dados em dispositivos Android, com foco em aplicações relacionadas a redes móveis.

#### Objetivo
Investigar e implementar o uso de parâmetros do ADB para a criação de ambientes de coleta de dados, permitindo análises de desempenho, conectividade e comportamento de redes móveis.

#### Contexto
O projeto envolve uma etapa teórica, baseada em revisão da literatura, e uma etapa prática, que inclui a configuração de ambientes, desenvolvimento de scripts e validação em cenários reais.

#### Organização do Repositório
O repositório será estruturado para conter:
- Scripts de automação da coleta de dados
- Registros de logs coletados
- Documentação técnica e guias de uso
- Materiais de apoio para análise dos dados

#### Status do Projeto
Em desenvolvimento – fase prática em andamento.
